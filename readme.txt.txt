MTG Card Search API


This is used to search through a database of mtg cards



Services to create:
	Batch sorter: Inputs a list of card names, then filters out them based on the user's search preferences
	set api: used for the sets of cards
	card api: used to get lists of mtg cards
	combo api: used to get a list of all interactions between cards for combos / synergies




Rescources:
Ruby 2.2
Rails 4.2
MTGJson - v3.5.1
SQLite3
